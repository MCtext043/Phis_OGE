# Сайт подготовки к ОГЭ по физике (Flask + Bootstrap)

Небольшой справочник по разделам физики для ОГЭ: формулы, учёные и тест на закрепление знаний.

## Требования

- Python 3.10+ (лучше 3.11+)

## Установка и запуск (Windows / PowerShell)

1. Перейдите в папку проекта:

   `cd "C:\Users\user\PycharmProjects\Physics OGE"`

2. Создайте виртуальное окружение (если его ещё нет):

   `python -m venv .venv`

3. Активируйте окружение:

   `.venv\Scripts\Activate.ps1`

4. Установите зависимости:

   `pip install -r requirements.txt`

5. Запустите сервер:

   `python -m flask --app app run --debug --port 5000`

6. Откройте в браузере:

   `http://127.0.0.1:5000`

## Структура

- `app.py` — логика сайта и данные справочника/теста (50 вопросов, в попытке 5 случайных)
- `templates/` — HTML-шаблоны (Bootstrap)
- `static/` — стили (серый фон, синий акцент)
